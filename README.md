*Summary of Features

Multi-page static website

Responsive layout (desktop/mobile)

Interactive forms with validation

Dynamic content using JavaScript

Image gallery / modal (if included)

*JavaScript Explanation

DOM selection using getElementById / querySelector

Event handling with addEventListener

Input validation using .trim() and .length

Conditional logic for user feedback

Optional use of localStorage (if implemented)

*Known Bugs

Form allows submission with disabled JS

Layout may break on very small screens (<360px)

No backend: data not persisted after refresh

Browser-dependent styling differences

*How to Run the Site

Download or clone the project

Open index.html in a web browser

No server or installation required
